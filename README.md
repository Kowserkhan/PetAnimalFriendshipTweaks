# Pet and Animal Friendship Tweaks

**Description**

This mod allows you to configure the heart limits of your pets and farm animals. Pet and farm animal friendship logic works differently,  
each heart is 200 points with a max of 1000 (5 hearts). This mod allows it to go to upto a max of 3000 (15 hearts). This is the source code.

---

## Features

- Customise the maximum friendship points for **pets** and **farm animals**.
- Raise the cap from the vanilla 1000 points (5 hearts) up to **3000 points (15 hearts)**.

---

## Source Code

This repository contains the full source code. You are free to build, modify, and contribute.

**Build instructions** (if you want to compile from source):

- Open the solution in JetBrains Rider (or your preferred C# IDE).
- Restore NuGet packages.
- Build the project – the output mod will be placed in the `bin/net6.0` folder as zip and to your `game/mods` folder like a normal mod if you have gamepath specified in the project file.
